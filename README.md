# Personal Expenses Tracker 

## Utilizare
-Adauga cheltuieli: Introducerea sumei, a categoriei (ex: mancare, transport, rate) si inregistrarea automata a datei. Salveaza datele intr-un fisier .json. 

-
## Rulare program
- Descarca 'main.py' intr-un folder.
- Deschide un terminal sau o consola in folderul unde se afla main.py .
- Scrie in terminal: python main.py
